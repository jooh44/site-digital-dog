## Story

**As a** website visitor,  
**I want** to see a carousel showcasing Digital Dog's portfolio of created websites,  
**so that** I can see examples of projects and build trust in the company's capabilities.

---

## Acceptance Criteria

1. Portfolio carousel section implemented on homepage
2. Carousel displays 5 portfolio items
3. Each item shows: Category, Name, Description, Placeholder icon, Link (optional)
4. Auto-play functionality (5 seconds interval)
5. Pause auto-play on user interaction (swipe, button click, dot click)
6. Swipe gestures work on mobile
7. Navigation arrows and dots indicators
8. Responsive design
9. Design matches design system with gradient accents

### Portfolio Section Specifications

**Component Structure:**
- Section header: "Nossos Sites Criados" with gradient accent on title
- Carousel container with fixed height
- Each slide shows:
  - Placeholder icon (rotating: Monitor, Code, Globe, Zap)
  - Category tag (with gradient)
  - Project name
  - Description
  - Optional "Visitar Site" link

**Technical Implementation:**
- Native React touch events (no external swipe library)
- Simple CSS transitions (no complex Framer Motion animations)
- Auto-play with intelligent pause/resume
- Cleanup of all timers on unmount

**Design Details:**
- Gradient accents: `from-[#ff6b35] via-[#ff1744] to-[#ff6b35]`
- Category tags use gradient text
- Placeholder icons with subtle background pattern
- "Preview em breve" badge
