## Story

**As a** website visitor,  
**I want** to see a consistent header and footer on all pages,  
**so that** I can navigate easily and access important links and information.

1. Header component created with navigation
2. Footer component created with all sections
3. Header and footer displayed globally (layout.tsx)
4. All links functional
5. Newsletter signup in footer
6. Responsive design (mobile menu)
