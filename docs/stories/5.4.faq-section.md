## Story

**As a** website visitor,  
**I want** to see frequently asked questions with answers,  
**so that** I can quickly find answers to common questions.

---

## Acceptance Criteria

1. FAQ section implemented on homepage
2. Accordion component created
3. 8-10 questions with answers
4. Schema markup for SEO (FAQPage)
5. Keyboard navigation support
6. Accessible (WCAG 2.1 AA)

  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Question text",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Answer text"
      }
    }
  ]
}
