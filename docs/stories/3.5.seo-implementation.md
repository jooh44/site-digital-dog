## Story

**As a** search engine,  
**I want** to properly index all pages with correct metadata and schema,  
**so that** the website appears in search results with rich snippets.

---

## Acceptance Criteria

1. Unique meta tags per page (title, description)
2. Open Graph tags implemented
3. Twitter Cards implemented
4. Sitemap.xml generated
5. Robots.txt configured
6. Schema.org markup (Organization, Article, FAQ)

```typescript
export const metadata: Metadata = {
  title: 'Page Title',
```

### Sitemap Generation
[Source: architecture/technology-stack.md]

Next.js 14 supports `app/sitemap.ts`:
```typescript
export default function sitemap() {
  return [
    {
      url: 'https://digitaldog.pet',
      lastModified: new Date(),
      changeFrequency: 'yearly',
      priority: 1,
    },
    // ... more pages
  ];
}
```

### Robots.txt
[Source: architecture/technology-stack.md]

Next.js 14 supports `app/robots.ts`:
```typescript
export default function robots() {
  return {
    rules: {
      userAgent: '*',
      allow: '/',
    },
    sitemap: 'https://digitaldog.pet/sitemap.xml',
  };
}
```
