§ 02

Botões

Sistema com 4 variantes — contenção como regra, gradiente como exceção
Primary — gradiente Fred

Uso: CTA principal da página. Máximo 1 por viewport. Sem glow default — apenas no hover com box-shadow: 0 0 32px rgba(255,107,53,0.3).

Ghost — borda sutil

Uso: ações secundárias, nav links, "Ver portfólio". Border rgba(255,255,255,0.12) no default. Sem background fill — só borda fica mais visível no hover.

Cyan outline — identidade

Uso: nav CTA (sutil), botões de ação secundária com identidade de marca. Aplicado com parcimônia — cyan como acento, não como cor de fundo.

Text + link — inline actions
Ver portfólio → ← Voltar Saiba mais Fale agora →

Uso: ações terciárias, links de navegação in-page, pós-submit WhatsApp link. "Fale agora →" em gradiente Fred — único texto colorido da página.

Disabled + Loading states

Disabled: background quase invisível, texto 0.25 de opacidade. Loading: spinner + texto — sem skeleton, sem barra de progresso no botão.
